# spark_samples

Samples for https://github.com/pxscene/pxCore

## lightning
  * application-texture
  * media-player
  * performance-test-unlimited

To build lightning sample:
```
npm install -g git://github.com/WebPlatformForEmbedded/Lightning-CLI.git#feature/standalone-dist-command
cd <app folder>
lng dist --spark
lng dist #es6
```
